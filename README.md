**DryShades** is a mobile application designed to simplify the process of buying, selling, and renting residential and commercial properties. The app connects property owners and renters directly, reducing time, cost, and effort involved in traditional house-hunting methods.

This project was developed as a **Third Year Project (CSE-3300)** for the Department of Computer Science & Engineering, Leading University.

---

## ğŸ“Œ Project Overview

Finding rental housing is often time-consuming and inefficient for both renters and property owners. DryShades addresses this problem by providing a centralized, easy-to-use mobile platform where users can:

- Browse properties by category  
- Contact owners directly  
- Save favorite listings  
- Post property advertisements  

The application is built using **Flutter** and **Firebase**, ensuring cross-platform compatibility and real-time data handling.
